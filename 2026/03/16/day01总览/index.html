3.д»Јз ҒејҖеҸ‘

жіЁи§Ј@RequestBody

  1. @RequestBody жҳҜ Spring MVC жЎҶжһ¶дёӯзҡ„дёҖдёӘйҮҚиҰҒжіЁи§ЈпјҢдё»иҰҒз”ЁдәҺеӨ„зҗҶ HTTP иҜ·жұӮдҪ“дёӯзҡ„еҶ…е®№пјҢе°Ҷе…¶иҪ¬жҚўдёә Java еҜ№иұЎгҖӮ

  2. ж”ҜжҢҒзҡ„ Content-Type

  • application/json (жңҖеёёз”Ё)
  • application/xml
  • text/plain
  • multipart/form-data
  1. @RequestBody жҳҜ Spring MVC дёӯеӨ„зҗҶиҜ·жұӮдҪ“зҡ„ж ёеҝғжіЁи§ЈпјҢе®ғпјҡ
  • иҮӘеҠЁе°Ҷ JSON/XML иҪ¬жҚўдёә Java еҜ№иұЎ
  • ж”ҜжҢҒеӨҚжқӮзҡ„ж•°жҚ®з»“жһ„е’ҢеөҢеҘ—еҜ№иұЎ
  • дёҺйӘҢиҜҒжЎҶжһ¶е®ҢзҫҺйӣҶжҲҗ
  • жҸҗй«ҳдәҶејҖеҸ‘ж•ҲзҺҮе’Ңд»Јз ҒеҸҜиҜ»жҖ§

жҺҢжҸЎ @RequestBody зҡ„дҪҝз”ЁеҜ№дәҺејҖеҸ‘зҺ°д»Ј RESTful API иҮіе…ійҮҚиҰҒгҖӮ

жіЁи§Ј@Bean

  1. @Bean жҳҜ Spring жЎҶжһ¶дёӯзҡ„дёҖдёӘйҮҚиҰҒжіЁи§ЈпјҢз”ЁдәҺж–№жі•зә§еҲ«зҡ„еЈ°жҳҺгҖӮе®ғе‘ҠиҜү Spring е®№еҷЁпјҡвҖқиҝҷдёӘж–№жі•е°Ҷиҝ”еӣһдёҖдёӘеҜ№иұЎпјҢиҜҘеҜ№иұЎеә”иҜҘиў«жіЁеҶҢдёә Spring еә”з”ЁдёҠдёӢж–Үдёӯзҡ„ BeanгҖӮвҖқ

  2. @Bean vs @Component

зү№жҖ§ @Bean @Component
дҪҝз”ЁдҪҚзҪ® ж–№жі•зә§еҲ« зұ»зә§еҲ«
жҺ§еҲ¶жқғ е®Ңе…ЁжҺ§еҲ¶е®һдҫӢеҢ–иҝҮзЁӢ з”ұ Spring жҺ§еҲ¶е®һдҫӢеҢ–
з¬¬дёүж–№еә“ йҖӮеҗҲй…ҚзҪ®з¬¬дёүж–№зұ» йҖӮеҗҲиҮӘе·ұзј–еҶҷзҡ„зұ»
зҒөжҙ»жҖ§ жӣҙй«ҳпјҢеҸҜд»ҘеҢ…еҗ«еӨҚжқӮйҖ»иҫ‘ зӣёеҜ№з®ҖеҚ•
  1. жңҖдҪіе®һи·ө
  • еңЁ @Configuration зұ»дёӯдҪҝз”ЁпјҡзЎ®дҝқ Bean иў«жӯЈзЎ®д»ЈзҗҶ
  • е‘ҪеҗҚи§„иҢғпјҡдҪҝз”ЁжҳҺзЎ®зҡ„ Bean еҗҚз§°
  • йҒҝе…ҚеүҜдҪңз”Ёпјҡ@Bean ж–№жі•еә”иҜҘжҳҜе№Ӯзӯүзҡ„
  • еҗҲзҗҶдҪҝз”ЁжқЎд»¶жіЁи§Јпјҡж №жҚ®зҺҜеўғжқЎд»¶еҲӣе»ә Bean
  1. @Bean жіЁи§ЈжҳҜ Spring жЎҶжһ¶дёӯйқһеёёејәеӨ§зҡ„е·Ҙе…·пјҢе®ғжҸҗдҫӣдәҶпјҡ
  • зҒөжҙ»зҡ„ Bean е®ҡд№үж–№ејҸ
  • е®Ңж•ҙзҡ„е®һдҫӢеҢ–иҝҮзЁӢжҺ§еҲ¶
  • ж–№дҫҝзҡ„дҫқиө–з®ЎзҗҶ
  • жқЎд»¶еҢ– Bean еҲӣе»әиғҪеҠӣ
  • з¬¬дёүж–№еә“йӣҶжҲҗж”ҜжҢҒ

жҺҢжҸЎ @Bean жіЁи§Јзҡ„дҪҝз”ЁпјҢиғҪеӨҹи®©дҪ жӣҙзІҫз»Ҷең°жҺ§еҲ¶ Spring е®№еҷЁзҡ„иЎҢдёәпјҢзј–еҶҷеҮәжӣҙеҠ зҒөжҙ»е’ҢеҸҜз»ҙжҠӨзҡ„еә”з”ЁзЁӢеәҸгҖӮ

Swagger

Swagger жҳҜдёҖдёӘи§„иҢғе’Ңе®Ңж•ҙзҡ„жЎҶжһ¶пјҢз”ЁдәҺз”ҹжҲҗгҖҒжҸҸиҝ°гҖҒи°ғз”Ёе’ҢеҸҜи§ҶеҢ– RESTful йЈҺж јзҡ„ Web жңҚеҠЎ(https://swagger.io/)гҖӮ е®ғзҡ„дё»иҰҒдҪңз”ЁжҳҜпјҡ

  • дҪҝеҫ—еүҚеҗҺз«ҜеҲҶзҰ»ејҖеҸ‘жӣҙеҠ ж–№дҫҝпјҢжңүеҲ©дәҺеӣўйҳҹеҚҸдҪң
  • жҺҘеҸЈзҡ„ж–ҮжЎЈеңЁзәҝиҮӘеҠЁз”ҹжҲҗпјҢйҷҚдҪҺеҗҺз«ҜејҖеҸ‘дәәе‘ҳзј–еҶҷжҺҘеҸЈж–ҮжЎЈзҡ„иҙҹжӢ…
  • еҠҹиғҪжөӢиҜ•

Springе·Із»Ҹе°ҶSwaggerзәіе…ҘиҮӘиә«зҡ„ж ҮеҮҶпјҢе»әз«ӢдәҶSpring-swaggerйЎ№зӣ®пјҢзҺ°еңЁеҸ«SpringfoxгҖӮйҖҡиҝҮеңЁйЎ№зӣ®дёӯеј•е…ҘSpringfox пјҢеҚіеҸҜйқһеёёз®ҖеҚ•еҝ«жҚ·зҡ„дҪҝз”ЁSwaggerгҖӮ
